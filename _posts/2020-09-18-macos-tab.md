## 背景
&emsp;&emsp;最近公司配了台 mac mini，第一次使用这个系统的终端，发现无法使用 tab 键对文件名进行自动补全。         

<br><br>

## 使用技巧
###### 终端自动补全的配置  
1.打开终端，输入: `$ nano .inputrc`           
2.在文件里面写上：             
```
set completion-ignore-case on

set show-all-if-ambiguous on

TAB: menu-complete
```
3.使用组合键 `ctrl + o`，然后回车                     
4.重启终端，自动补全按 tab 键即可
