## 1 标识符
### 1.1 标识符
&emsp;&emsp;标识符即程序员日常定义的类名、方法名、变量名、包名......       

<br><br>


### 1.2 关键字 
&emsp;&emsp;关键字即 java 内部已使用的标识符且均有自己的特殊含义和用途，因此用户不能定义与关键字完全同名的标识符。    

<br><br>


### 1.3 保留字
&emsp;&emsp;保留字即目前未成为但日后可能会成为关键字的标识符，因此用户不能定义与保留字完全同名的标识符。

<br><br>


### 1.4 命名规范  
1. 包名：所有单词所有字母均小写   
2. 类名、方法名：所有单词的首字母均大写   
3. 变量名、方法名：小驼峰 xxxYyyZyy    
4. 常量名：XXX_YYY_ZZZ        

<br><br><br>
